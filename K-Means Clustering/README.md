#  K-Means Clustering: Customer Segmentation Project

## Project Overview
This project was my hands-on exploration of **K-Means Clustering**, one of the most popular unsupervised learning algorithms. The goal was simple but meaningful: group customers into segments based on their behavior and numerical characteristics. Instead of predicting an outcome, I wanted to uncover natural patterns in the data and see how customers cluster together.

## About the Dataset
The dataset I worked with is a synthetic customer segmentation dataset that includes a mix of useful numerical features such as:

- Spending patterns
- Income levels
- Various feature measurements that describe customer behavior

Even though the dataset is synthetic, it reflects the structure of real-world segmentation problems used in marketing and retail.
